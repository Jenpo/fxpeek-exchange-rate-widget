# Publishing Checklist

Repository name:

`fxpeek-exchange-rate-widget`

Suggested description:

`Embeddable historical exchange rate widget for blogs, dashboards, and spreadsheet workflows.`

Suggested topics:

`exchange-rates`, `currency`, `widget`, `iframe`, `finance`, `csv`, `api`, `spreadsheet`

After publishing:

1. Confirm README contains the live widget page, embed script, API docs, and annual report links.
2. Confirm `examples/basic-embed.html` references `https://fxpeek.com/embed/fxpeek-widget.js`.
3. Add the published URL to `publication-tracker.csv`.
4. Run `npm run offsite:check` and `npm run offsite:report`.
5. Use the repo link in guest post pitches to finance/accounting/spreadsheet blogs.
